# Microsoft Azure
Aquí explico cómo se creó la cuenta de Microsoft Azure donde desplegamos nuestra aplicación web.


## Crear cuenta

1. Primero me dirijo al Portal de Azure for Students y se le da a empezar gratis.
2. Inicio sesión con mi correo institucional.
3. Realizamos algunas verificaciones por seguridad.
4. Verificamos número telefónico.

Con esto tendremos una cuenta de Microsoft Azure para estudiantes la cual tendrá 100 créditos gratis para desplegar proyectos o para muchas otras opciones más.
